for documents designed to be displayed in a web browser. It defines the

CSS
CSS stands for Cascading Style Sheets. It is a stylesheet language used
to describe the presentation of a document written in HTML or XML.
CSS describes how elements should be rendered on screen, on paper, in
speech, or on other media.
Javascript
JavaScript, often abbreviated as JS, is a programming language that is
one of the core technologies of the World Wide Web, alongside HTML and
CSS. As of 2023, 98.7% of websites use JavaScript on the client side for
webpage behavior, often incorporating third-party libraries.
